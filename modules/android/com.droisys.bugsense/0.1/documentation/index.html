bugsense Module

Description

This Module is for integrating Bugsense which helps for testing of the app either during development or after deployment to market. Since Titanium handles most of the Runtime Error, this module will help only when app force closes.

Accessing the bugsense Module

To access this module from JavaScript, you would do the following:

var bugsense = require("com.droisys.bugsense");

The bugsense variable is a reference to the Module object.

Reference

startTracking(String id);

Set your API KEY of Bugsense Account and Starts tracking

Usage

bugsense.startTracking("d5c9c043");
