bugsense Module

Description

An Module for Bugsense Tracking Tool. Go here to know more http://www.bugsense.com/. Register an API Key to use.

Accessing the bugsense Module

To access this module from JavaScript, you would do the following:

var bugsense = require("com.droisys.bugsense");

The bugsense variable is a reference to the Module object.

Reference

Setting Up the Module is very simple.

_PROJECTNAMEASIDENTIFIER.function

bugsense.setup('YOUR-API-KEY');
